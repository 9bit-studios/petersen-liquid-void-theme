## 🌍 International Shipping Details

### Customs & Duties

**What You Need to Know:**
- Import duties vary by country and product value
- Customer is responsible for all customs fees
- We mark packages with accurate commercial value
- Duty-free thresholds vary by destination

**Common Duty Rates:**
- **Canada:** 0-6.5% + GST/HST
- **EU Countries:** 0-4.7% + VAT (varies)
- **UK:** 0-4.7% + 20% VAT
- **Australia:** 5% + 10% GST

### Restricted Items

**Cannot Ship To:**
- Countries with current trade restrictions
- Military APO/FPO addresses (contact for alternatives)
- Remote islands without reliable postal service

**Special Considerations:**
- Some miniatures require special documentation
- Large orders may require commercial import licenses
- Contact support for orders over $1000 USD

## 📱 Tracking & Delivery

### Tracking Your Order

**How to Track:**
1. Check shipping confirmation email
2. Use tracking number on carrier website
3. Enable SMS/email notifications from carrier
4. Contact support if no updates after 24 hours

**Tracking Updates:**
- **Order Confirmed:** Payment processed
- **Preparing:** Items being picked and packed
- **Shipped:** Package handed to carrier
- **In Transit:** En route to destination
- **Out for Delivery:** Arriving today
- **Delivered:** Package received

### Delivery Options

**Signature Requirements:**
- Orders over $200 require signature
- Large miniature sets require signature
- International orders typically require signature
- No signature delivery available upon request

**Safe Delivery:**
- Carriers authorized to leave packages
- We’re not responsible for stolen packages
- Consider delivery to workplace for security
- Package insurance covers loss/damage

- Update delivery instructions

## 📞 Shipping Support

### Before You Ship Returns

**Contact Us First:**
- Get return authorization number
- Confirm return shipping address
- Understand return shipping responsibilities
- Ask about expedited replacement options

### Shipping Damage Claims

**Report Immediately:**
1. Take photos of damaged package and contents
2. Keep all packaging materials
3. Contact support within 48 hours
4. File claim with carrier if required

### Lost Package Process

**If Your Package is Lost:**
1. Wait 7 days past expected delivery
2. Check with neighbors and building management
3. Contact carrier to file lost package claim
4. We’ll process replacement once claim confirms loss
