## 📚 Portal Navigation Guide

### Quick Start Guide

#### First Time Access
1. **Login**: Use your organization email at notion.so
2. **Find Your Hub**: 
   - Petersen Games team → Petersen Games Hub
   - 9Bit Studios team → 9Bit Studios Hub
3. **Dashboard**: Your personalized dashboard loads automatically
4. **Customize**: Click ⚙️ to personalize your view

#### Essential Navigation

##### Keyboard Shortcuts
- `Cmd/Ctrl + K`: Quick search anything
- `Cmd/Ctrl + P`: Open page finder
- `Cmd/Ctrl + /`: Show all commands
- `Space`: Quick preview in databases
- `Esc`: Exit any modal or view

##### Siri Voice Commands (Mac/iOS)
- "Hey Siri, show my active campaigns in Notion"
- "Hey Siri, create a new content draft"
- "Hey Siri, what's my team's workload today?"

## 🎮 Petersen Games Campaign Management Guide

### Campaign Lifecycle Management

#### 1. Creating a New Campaign
```
Step 1: Navigate to Campaign Master
Step 2: Click "+ New Campaign"
Step 3: Fill required fields:
   - Campaign Name
   - Platforms (select all that apply)
   - Start/End Dates
   - Budget
   - Objectives
Step 4: AI Assistant will analyze and suggest optimizations
Step 5: Save and the system creates related tasks automatically
```

#### 2. Content Planning & Scheduling

##### Content Calendar Workflow
1. **View Calendar**: Switch to Calendar view in Content Library
2. **Create Content**: 
   - Click on date to create
   - AI suggests optimal posting times
   - Brand validator checks consistency
3. **Review Process**:
   - Draft → AI Review → Human Review → Approved → Scheduled
4. **Multi-Platform Publishing**:
   - Select platforms in content item
   - System adapts content for each platform
   - Preview before publishing

##### AI Content Enhancement
- **Quality Score**: Aim for 85%+ before publishing
- **Brand Alignment**: Must be 90%+ for approval
- **SEO Optimization**: Follow AI suggestions for keywords
- **Readability**: Target score based on audience

#### 3. Campaign Analytics

##### Performance Metrics
- **Real-time Updates**: Data refreshes every hour
- **ROI Calculation**: Automatic based on conversions
- **Engagement Tracking**: Cross-platform aggregation
- **AI Insights**: Daily recommendations for optimization

##### Reading the Dashboard
```
🟢 Green metrics = Above target
🟡 Yellow metrics = Near target  
🔴 Red metrics = Need attention
📈 Trending up = Positive momentum
📉 Trending down = Review needed
```

#### Member Tiers Dashboard

## 💻 Development Workflow Guide

### Project Management

#### Sprint Planning
1. **Sprint Board Setup**:
   - Two-week sprints by default
   - Drag tasks between columns
   - Priority indicators (🔴 High, 🟡 Medium, 🟢 Low)

2. **Task Assignment**:
   - Click assignee field
   - System suggests based on workload
   - Automatic notification to assignee

3. **Progress Tracking**:
   - Update status by dragging
   - Log time in "Actual Hours"
   - Add comments for blockers

#### Development Pipeline

##### Stages Explained
- **Planning**: Requirements gathering, design
- **Development**: Active coding/building
- **Testing**: QA and bug fixes
- **Staging**: Pre-production testing
- **Production**: Live and monitored

##### Best Practices
1. **Daily Updates**: Move tasks daily
2. **Time Logging**: Helps future estimates
3. **Documentation**: Update as you go
4. **Code Reviews**: Link PR in comments

### Integration with Git
```bash
# Link commits to tasks
git commit -m "feat: [TASK-ID] Add new feature"

# Auto-updates appear in Notion
```

## 🆘 Troubleshooting & Support

### Common Issues

#### Sync Problems
- **Solution**: Refresh page (Cmd/Ctrl + R)
- Check internet connection
- Clear browser cache
- Try different browser

- Verify you're in right workspace
